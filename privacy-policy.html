Nebulae

Privacy Policy

Effective date: July 4, 2026

Overview

Nebulae is a personal finance tracking app designed to help you record income, expenses, transactions, and view your balance.

Data Storage

Income and expense entries are stored locally on your device using SQLite. In the current version, Nebulae does not transmit your finance records to Nebulae developer servers.

Guest Mode

Nebulae currently supports Continue as Guest. Guest mode does not create an online account.

No Cloud Sync Yet

The current version does not sync finance data to Nebulae servers, Supabase, or any cloud service.

No Ads or Analytics

The current version does not intentionally include ads or analytics SDKs.

Export

CSV export is manually triggered by you and is intended for viewing or sharing your records. JSON backup is intended for restoring app data. You are responsible for where exported files are shared, saved, or stored after export.

Local Backup & Restore

Nebulae lets you create a local JSON backup file when you choose Create Backup. The backup file may include income and expense transactions entered by you. You choose where to save or share the backup file, such as device files, Google Drive, email, or messaging apps.

Nebulae does not automatically upload backup files to Nebulae servers. The current version has no cloud sync. Please keep backup files private and secure.

Delete and Restore Data

The Clear All Data option permanently deletes local finance records from your device. Restore Backup reads a selected Nebulae backup file and can replace your current local transactions after confirmation.

Permissions

Nebulae does not need location, camera, microphone, contacts, SMS, or phone access for its current features.

Future Features

Google login, Apple login, cloud sync, analytics, crash reporting, ads, or AI features may be added later only with policy updates before release.

Contact

Contact support details will be added before public release.
